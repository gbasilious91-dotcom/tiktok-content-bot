Privacy Policy

Last updated: May 21, 2026

This Privacy Policy describes how TikTok Content Bot ("the App", "we", "us") handles information when you use our application.

1. About This App

TikTok Content Bot is a personal single-account automation tool that generates and publishes short-form video content to a single TikTok account. It is not a multi-user platform and does not collect, store, or process data from third-party users.

2. Information We Access

The App accesses the following information through TikTok's API solely to enable posting functionality:

  • TikTok Account ID (open_id): Used to identify the account to post content to.
  • Basic Profile Info: Accessed via the user.info.basic scope to confirm account identity.
  • Video Upload Access: Used via video.upload and video.publish scopes to post AI-generated videos to the account.

3. Data Storage

The App stores only the OAuth access token locally on the developer's own machine in a .env file. No data is stored on external servers, databases, or shared with third parties.

4. Data Sharing

We do not sell, trade, or share any data with third parties. All API credentials and tokens are stored locally and used solely for the purpose of posting content to the authorized TikTok account.

5. Third-Party Services

The App uses the following third-party APIs to generate content:

  • Anthropic Claude API — for script generation
  • OpenAI API — for text-to-speech voiceover
  • Replicate API — for AI image generation
  • TikTok Content Posting API — for video publishing

Each of these services has their own privacy policies governing data use.

6. Children's Privacy

This App is not directed at children under 13 and does not knowingly collect information from children.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated date.
